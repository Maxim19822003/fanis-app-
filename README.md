# 🤖 Фанис — PWA Техподдержка

Техническое приложение для экспедиторов. Работает на телефоне как сайт и устанавливается как приложение.

---

## 📱 Как установить (с телефона)

### Шаг 1: Загрузить на GitHub
1. Откройте [github.com](https://github.com) в браузере телефона
2. Войдите в аккаунт (или создайте)
3. Нажмите **+** → **New repository**
4. Название: `fanis-app`
5. Нажмите **Create repository**
6. Нажмите **Add file** → **Upload files**
7. Выберите файлы: `index.html`, `style.css`, `app.js`, `sw.js`, `manifest.json`
8. Нажмите **Commit changes**

### Шаг 2: Включить GitHub Pages
1. В репозитории нажмите **Settings** (внизу, шестерёнка)
2. Прокрутите вниз до **Pages**
3. В разделе **Source** выберите ветку `main`
4. Нажмите **Save**
5. Через 2-3 минуты появится ссылка вида `https://ваш-ник.github.io/fanis-app`

### Шаг 3: Открыть и установить
1. Откройте ссылку в Chrome (Android) или Safari (iPhone)
2. **Android**: нажмите ⋮ → "Добавить на главный экран"
3. **iPhone**: нажмите Поделиться → "На экран Домой"

## ⚙️ Как редактировать контент

### Вход в админку:
1. Откройте приложение
2. Нажмите **⚙️ Админ** внизу главного экрана
3. Введите пароль: `fanis2024`

### Добавить поломку:
1. В админке нажмите **🔧 Поломки**
2. Нажмите **➕**
3. Заполните поля:
   - **Название** — например "Монетник"
   - **Эмодзи** — например 💰
   - **Описание шагов** — каждый шаг с новой строки
   - **Ссылка на видео** — RuTube или VK
   - **Ссылка на картинку** — ссылка на мем (опционально)
   - **Доп. инфо** — фишка/подсказка (опционально)
   - **Контакт** — кого звонить (Фанис или Вильдан)
   - **Активно** — показывать на сайте или нет
4. Нажмите **💾 Сохранить**

### Добавить код ошибки:
1. В админке нажмите **⚠️ Коды ошибок**
2. Нажмите **➕**
3. Заполните поля:
   - **Система** — UMAR / CLE / UMAR_new / CLE_new
   - **Код** — например ER 01
   - **Описание** — что означает ошибка
   - **Период** — когда появляется
   - **Решение** — шаги с новой строки
   - **Картинка** — ссылка на мем (опционально)
   - **Контакт** — кого звонить
4. Нажмите **💾 Сохранить**

### Изменить контакты:
1. В админке нажмите **📞 Контакты**
2. Введите номера телефонов
3. Нажмите **💾 Сохранить**

## 🎬 Как добавить видео

### RuTube:
1. Загрузите видео на [rutube.ru](https://rutube.ru)
2. Откройте видео
3. Скопируйте ссылку из адресной строки (вида `https://rutube.ru/video/abc123/`)
4. Вставьте в поле "Ссылка на видео" в админке

### VK Видео:
1. Загрузите видео в VK
2. Откройте видео
3. Скопируйте ссылку (вида `https://vk.com/video-123_456`)
4. Вставьте в поле "Ссылка на видео" в админке

---

## 🖼️ Как добавить картинку-мем

1. Загрузите картинку на [imgur.com](https://imgur.com) (не требует регистрации)
2. Откройте картинку
3. Нажмите ПКМ → "Копировать адрес изображения"
4. Вставьте ссылку в поле "Ссылка на картинку/мем"

---

## 💾 Резервное копирование

В админке нажмите **💾 Экспорт данных** — скачается JSON-файл со всеми данными.

Если нужно восстановить — нажмите **📥 Импорт данных** и вставьте содержимое JSON-файла.
